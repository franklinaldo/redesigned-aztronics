    • The Raspberry Pi 4 Single Board Computer 4GB is the latest product from Raspberry Pi. The speed and performance of the new Raspberry Pi 4 is a step up from earlier models. For the first time, Raspberry Pi has built a complete desktop experience. Whether you’re editing documents, browsing the web with a bunch of tabs open, juggling spreadsheets or drafting a presentation, you’ll find the experience smooth and very recognisable — but on a smaller, more energy-efficient and much more cost-effective machine. With the new model you can run two monitors at once in 4K. With the added feature of no fans the Raspberry Pi 4 is energy-efficient and runs silently and uses far less power than other computers. Another new feature is the upgraded USB capacity: along with two USB 2 ports you’ll find two USB 3 ports, which can transfer data up to ten times faster.
